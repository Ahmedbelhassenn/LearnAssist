# LearnAssist

Platforme d'apprentissage en ligne (LMS) connectant Formateurs et Participants.

## Structure du projet

Ce dépôt est un **Monorepo** contenant :

*   **Frontend** : Application Angular 18 (`/Learn_Assist_Frontend`)
*   **Backend** : Application Spring Boot 3.4.2 (`/Learn_Assist_Backend`)

## Installation

### Frontend
\`\`\`bash
cd Learn_Assist_Frontend/LearnAssist
npm install
ng serve
\`\`\`

### Backend
\`\`\`bash
cd Learn_Assist_Backend
./mvnw spring-boot:run
\`\`\`
